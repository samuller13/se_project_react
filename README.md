# What to wear

This is a full-stack web application called "WTWR" ("What to wear"). It will read weather data from a Weather API and then recommend suitable clothing to the user based on that data.

## Tech Stack

- HTML
- CSS
- Responsive Design
- JavaScript
- React
- JSX
